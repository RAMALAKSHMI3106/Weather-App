#  Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript. It provides real-time weather information for any city using the OpenWeather API.

##  Features
- Search weather by city name
- Displays temperature, humidity, and wind speed
- Dynamic weather icons
- Responsive design

##  Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
